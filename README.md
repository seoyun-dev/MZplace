# PROJECT: SeoulWalk
## 소개
- 서울의 다양한 실외, 실내활동을 한눈에 보여주는 사이트
    - 다양한 활동 카테고리 제시
    - 위치 기반 가까운 장소 추천 기능
    - 장소 및 코스 리뷰 기능
    - 사용자의 찜을 기반으로 맞춤 추천하는 ML 기능
        - UBCF, IBCF 알고리즘
    
- FE [Github](https://github.com/JJongsKim/Seoul-Walk)
