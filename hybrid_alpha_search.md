# Hybrid Alpha Search Results

## 데이터 구성
- 사용 데이터: `places/hearts_data.csv`
- 평가 방식: 각 사용자마다 가장 최신 찜 1개를 테스트 정답으로 떼어내고, 나머지 찜 데이터를 학습 데이터로 사용한 Leave-One-Out 검증
- 따라서 평가 지표의 정답은 테스트 분할에서 제거한 실제 유저-아이템 상호작용(찜)입니다.

## Best alpha 선정 기준
- `alpha`는 hybrid 모델에서 UBCF와 IBCF 점수를 결합하는 가중치입니다.
- 본 보고서는 `NDCG` 기준으로 alpha를 탐색하여 최적 alpha를 선정했습니다.
- 실제 서비스 코드(`places/views.py`)에서도 동일한 score fusion 방식을 적용하여 best alpha=0.8을 사용합니다.
